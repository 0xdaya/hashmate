# Hashmate

Hasmate is a simple hash lookup utility written in python3

## Installation
```
git clone https://github.com/dayanandpathak/hashmate.git
```

## Usage:-

### For Windows:-

``` 
python3 hash.py
```

### For Linux:-

```
1. chmod 770 hash.py
2. ./hash.py
```
## This script is only compatible with Python 3.x
